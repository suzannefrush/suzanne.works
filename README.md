## Developing

To work locally pull this project, ensure you're using [fnm](https://github.com/Schniz/fnm) or [nvm](https://github.com/nvm-sh/nvm) to use the right Node version.

Run `npm install` to update and ensure dependencies are correct.

Run `npm run dev` to run a dev server. You can now navigate to [http://localhost:3000](http://localhost:3000) to view your the application running locally.

It is recommended you use [VSCode](https://code.visualstudio.com/) and [Claude Code](https://www.claude.com/product/claude-code) as pre-built configurations for both tools are present in this project. VSCode will recommend plugins based on this configuration, it is recommended you install those plugins.
